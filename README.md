## ⚡ What is MapReaper?

**MapReaper** is a lightweight browser extension designed to enhance GeoGuessr-style gameplay.

It helps players:
- 🔍 recognize location patterns faster  
- 🧭 analyze roads, signs, and environments  
- 🎯 improve accuracy and decision-making  
- ⚡ speed up overall gameplay  

> Built for competitive players. Optimized for clarity. Designed for performance.

## 📦 Installation

1. Go to **Releases**
2. Download latest version
3. Open Chrome / Chromium
4. Navigate to `chrome://extensions/`
5. Enable **Developer Mode**
6. Click **Load unpacked**
7. Select extension folder
